代码的提交：
仓库的main分支用来存放稳定的代码文件，个人在push时，一定要先建立一个自己的分支，建议以自己名称的首字母为分支名称
当大家完成自己的任务后，再一起评估代码，最后将稳定的版本和main分支合并
